## 📌 About

This is a simple yet fully functional To-Do application built with HTML, CSS, and JavaScript.  
Add tasks, mark them complete, and delete them — all in a clean, minimal UI.

---

## ✨ Features

- ➕ **Add tasks** instantly
- ✅ **Mark tasks** as complete
- 🗑️ **Delete tasks** individually
- 💾 Tasks stay during the session
- 📱 Fully **responsive** design
- 🎨 Clean minimal UI
