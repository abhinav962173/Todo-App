# Todo-App

A simple and efficient To-Do application that helps users manage daily tasks. You can add, delete, and track your tasks easily.

## Features:
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and responsive UI

## Tech Stack:
- HTML
- CSS
- JavaScript

## How to Use:
1. Enter a task in the input field
2. Click on "Add" button to add the task
3. Click on a task to mark it as completed
4. Click delete button to remove a task
